### Option 1: Print from Browser (Recommended)

1. **Open `menu-front.html` in Chrome or Edge**
2. Press `Ctrl + P` (or `Cmd + P` on Mac)

   - Paper size: **A4**

1. **Export both pages to PDF** using the method above
2. Take PDFs to a print shop (Vistaprint, local printer, etc.)
3. **Recommended specs:**
   - Paper: **300gsm silk or gloss**
   - Finish: **Laminated** (wipeable, durable)
   - Size: **A4** (210mm x 297mm)
